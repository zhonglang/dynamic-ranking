# 动态排行榜生成器 (Dynamic Ranking Generator)

一个纯前端的动态排行榜动画生成工具，专为短视频创作者、数据分析师和社交媒体达人设计。支持多种酷炫动画效果、视频录制导出，以及深度视觉定制。

## 🎬 核心功能

### 1. 动态动画系统
- **9种动画类型**：
  - 🎯 **挤压式**：经典排行效果，新条目滑入并挤压已有条目。
  - 💫 **淡入式**：优雅的透明度渐变展示。
  - 🚀 **横向滑入式**：从左侧动感滑入，充满现代感。
  - 🎪 **缩放弹跳式**：中心弹出并带有弹性动画，活泼有趣。
  - 🃏 **翻转卡片式**：精致的3D翻转卡片效果。
  - 🛗 **升降机式**：底部弹跳升起，趣味十足。
  - 🦅 **上方飞入式**：条目从上方一个个往下飞入，带有弹性效果。
  - 🧬 **故障闪烁**：赛博朋克风格的色彩分离与位置抖动效果。
  - 📡 **扫描光束**：带有动态光束扫描的条形图填充动画。
  - 🌊 **波动入场**：流畅的波浪起伏进场效果。
- **智能缓动控制**：基于 `requestAnimationFrame` 实现 30fps 以上的流畅体验，内置 `easeOutBack` 等专业缓动函数。

### 2. 深度视觉定制
- **科技感背景**：
  - 内置 **数字雨**、**几何网格**、**能量粒子** 和 **星光背景** 等多种科技元素。
  - 提供 **7种预设背景主题**（深色科技、科技蓝、紫、绿、红、赛博朋克、太空深蓝）。
  - 支持 **一键开关背景主题**，可快速切换极简风格。
- **自定义背景图**：
  - 支持上传本地图片作为背景，并可实时调节 **背景图透明度**。
- **条形图样式**：
  - **自动色彩系统**：前三名拥有专属 **金/银/铜渐变色** 与勋章（🥇🥈🥉）；其他排名由 HSL 算法自动生成随机渐变色。
  - **透明度阶梯**：根据数值大小自动计算透明度，强化排名对比。
- **文字与数值配置**：
  - 可自由配置 **柱形图文字颜色**（默认白色，且不受背景透明度影响）。
  - **显示数值开关**：可选择是否在条形图上显示具体数值。
  - **数值位置可调**：支持“末端”（条形图内部）和“末端后”（条形图外部）两种位置，并可精确调节外部偏移量。

### 3. 节日与互动特效
- **智能烟花系统**：
  - 自动检测前三名出现时机，触发绚丽的 **烟花爆炸效果**。
  - 可定制烟花持续时间、密度、碎片速度及核心亮点占比。
- **动态标题**：
  - 支持自定义标题内容、文字大小及文字颜色。

### 4. 数据输入与处理
- **多模式输入**：
  - **文本模式**：直接粘贴标准 JSON 格式数据。
  - **文件模式**：支持 JSON/TXT 文件拖拽或点击上传。
- **智能解析**：自动兼容 `name/label/title` 和 `value/score` 等多种字段名，支持对象数组或纯键值对对象格式。
- **内置预设**：默认内置“十二星座”数据，方便新手快速上手。

### 5. 视频导出与分享
- **Canvas 流录制**：基于 MediaRecorder API 实现高质量视频录制。
- **多格式导出**：优先导出 MP4 (H.264)，自动兼容 WebM 格式。
- **预览功能**：支持“预览动画”模式，不触发录制即可实时查看最终效果。

## 🛠 技术实现

- **纯前端架构**：无需后端服务器，所有计算与渲染均在浏览器本地完成，保护数据隐私。
- **Canvas 2D 绘图**：高性能条形图绘制、渐变处理及粒子特效系统。
- **响应式设计**：适配 PC 端左右分栏操作与移动端查看，完美支持 9:16 短视频比例。
- **零依赖**：仅使用原生 HTML5, CSS3, JavaScript (ES6+)，轻量且加载极快。

## 📁 文件结构
- `index.html`: 页面结构与 UI 配置面板。
- `css/style.css`: 样式定义、布局及 CSS 动画。
- `js/app.js`: 核心逻辑类 `DynamicRanking`，负责数据处理、动画渲染、烟花系统及视频录制。
