# 动态排行榜生成器

一个纯前端的动态排行榜动画生成工具，支持从小到大的动态排序动画展示。

## 📋 项目概述

**动态排行榜生成器** 是一个基于Web的工具，用于创建和展示动态排序动画的排行榜。项目采用纯前端技术实现，无需后端服务器，直接在浏览器中运行。

### 核心特性
- 🎬 **动态排序动画**：从第12名（最小值）到第1名（最大值）的逐个出现动画
- 🎨 **精美视觉效果**：前三名特殊样式、渐变色条形图、平滑过渡动画
- 📱 **响应式设计**：完美适配桌面和移动设备
- 🚀 **零依赖**：仅使用原生HTML5、CSS3、JavaScript
- 💾 **多数据格式支持**：JSON数组/对象格式，支持文件上传和文本输入

## ✨ 核心功能

### 数据输入功能
- **双模式数据输入**：
  - 📝 **文本输入**：直接粘贴JSON格式数据
  - 📁 **文件上传**：支持拖拽或点击上传JSON/TXT文件
- **智能数据解析**：
  - 支持两种JSON格式：数组格式 `[{name: "xxx", value: 123}]` 和对象格式 `{"name": 123}`
  - 自动识别字段别名（支持name/label/title，value/score）
  - 数据验证和过滤（过滤无效值和负值）

### 动画效果功能
- **动态排序动画**：
  - 从第12名（最小值）开始逐个向上添加
  - 每个条形图有从左到右的飞入动画效果
  - 可配置的动画间隔时间（0.1-2秒）
- **视觉设计**：
  - 前三名特殊样式：金牌🥇、银牌🥈、铜牌🥉
  - 条形图宽度根据数值比例自动计算
  - 随机颜色生成算法（HSL色彩空间）
  - 渐变背景和阴影效果

### 设置配置功能
- **排行榜标题**：可自定义显示标题
- **动画速度控制**：条形图间隔时间调节（0.1-2秒）
- **默认值预设**：合理的默认配置

### 用户界面功能
- **响应式设计**：适配各种屏幕尺寸
- **Tab切换界面**：清晰的数据输入方式切换
- **视觉反馈**：按钮悬停效果、文件上传状态提示、错误信息弹窗

## 🚀 使用方法

### 基本操作流程

1. **选择数据输入方式**：
   - **文本输入**：切换到"文本输入"标签页，在文本框中粘贴JSON数据
   - **文件上传**：切换到"文件上传"标签页，拖拽或点击上传JSON/TXT文件

2. **配置设置**：
   - 输入排行榜标题（可选）
   - 设置条形图间隔时间（0.1-2秒）

3. **运行动画**：
   - 点击"运行动画"按钮
   - 观看从小到大的动态排序过程

### 数据格式示例

```json
// 格式1：数组格式（推荐）
[
  {"name": "白羊座", "value": 100},
  {"name": "金牛座", "value": 200},
  {"name": "双子座", "value": 150},
  {"name": "巨蟹座", "value": 180},
  {"name": "狮子座", "value": 220}
]

// 格式2：对象格式
{
  "白羊座": 100,
  "金牛座": 200,
  "双子座": 150,
  "巨蟹座": 180,
  "狮子座": 220
}
```

### 支持的字段别名

- **名称字段**：`name`、`label`、`title`
- **数值字段**：`value`、`score`
