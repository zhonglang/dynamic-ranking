# 动态排行榜生成器 (Dynamic Ranking Generator)

一个纯前端的动态排行榜动画生成工具，专为短视频创作者、数据分析师和社交媒体达人设计。支持多种酷炫动画效果、视频录制导出，以及深度视觉定制。

## 🎬 核心功能

### 1. 动态动画系统
- **10种动画类型**：
  - 🎯 **挤压式**：经典排行效果，新条目滑入并挤压已有条目。
  - 💫 **淡入式**：优雅的透明度渐变展示。
  - 🚀 **横向滑入式**：从左侧动感滑入，充满现代感。
  - 🎪 **缩放弹跳式**：中心弹出并带有弹性动画，活泼有趣。
  - 🃏 **翻转卡片式**：精致的3D翻转卡片效果。
  - 🛗 **升降机式**：底部弹跳升起，趣味十足。
  - 🦅 **上方飞入式**：条目从上方一个个往下飞入，带有弹性效果。
  - 🧬 **故障闪烁**：赛博朋克风格的色彩分离与位置抖动效果。
  - 📡 **扫描光束**：带有动态光束扫描的条形图填充动画。
  - 🌊 **波动入场**：流畅的波浪起伏进场效果。
- **智能缓动控制**：基于 `requestAnimationFrame` 实现 30fps 以上的流畅体验，内置 `easeOutBack` 等专业缓动函数。

### 2. 深度视觉定制
- **科技感背景**：
  - 内置 **数字雨**、**几何网格**、**能量粒子** 和 **星光背景** 等多种科技元素。
  - 提供 **12种预设背景主题**：
    - 动态主题：深色科技、科技蓝、紫、绿、红、赛博朋克、太空深蓝
    - 静态主题：矩阵网格、电子电路、蓝图设计、深邃星云
  - 支持 **一键开关背景主题**，可快速切换极简风格（无主题模式）。
- **自定义背景图**：
  - 支持上传本地图片作为背景。
  - 提供 **实时透明度调节** 及 **一键清除背景图** 功能。
- **条形图样式**：
  - **自动色彩系统**：前三名拥有专属 **金/银/铜渐变色** 与勋章（🥇🥈🥉）；其他排名由 HSL 算法自动生成随机渐变色。
  - **透明度阶梯**：根据数值大小自动计算透明度，强化排名对比。
- **标题定制**：
  - 自由配置标题内容、文字大小、颜色（默认白色）。

### 3. 数值显示配置 (Value Display)
- **位置灵活控制**：
  - **条形内**：数值显示在条形图右侧内部。
  - **条形外**：数值显示在条形图右侧外部。
- **专属间距配置**：当选择“条形外”时，可自定义数值与条形图之间的精确距离。
- **智能显示规则**：支持“条形外配置仅影响非前两名”，确保头部排名视觉效果的独特性。
- **颜色自定义**：可独立设置数值文字颜色。

### 4. 增强特效系统
- **烟花特效**：
  - **灵活开启方式**：支持单选框（Radio）快捷开启或关闭。
  - **动态触发名次**：
    - **自适应数据量**：触发名次下拉框根据上传的数据项数量动态渲染。
    - **自定义阈值**：用户可指定从第几名开始触发烟花（默认前3名）。
  - **多维度自定义**：
    - **10级密度调节**：随密度增加火箭数量、粒子总数及持续时长。
    - **5种图案模式**：随机图案、经典圆形、浪漫心形、璀璨星形、爆炸散射。
- **随机红包特效**：
  - 支持开启/关闭随机红包掉落效果。
  - 包含多种红包类型：传统红包、福袋、金币、元宝。
  - 带有物理下落和旋转动画效果。
- **鲜花花瓣特效**：
  - 支持开启/关闭花瓣飘落效果。
  - 从屏幕两侧生成粉色系花瓣，带有自然的飘动和旋转效果。
- **弹幕功能**：
  - **默认不开启**：初始状态为关闭，用户可手动开启。
  - **多弹幕支持**：可输入多个弹幕内容，用空格分隔。
  - **自定义字体颜色**：支持选择弹幕文字颜色。
  - **字体大小调节**：可设置弹幕文字大小（12-36px）。
  - **自动滚动**：弹幕从屏幕左侧自动滚动到右侧。
  - **轮播效果**：弹幕会持续循环播放，速度适中。

### 5. 数据处理与导出
- **智能解析**：自动兼容多种字段名（name/value等），支持 JSON/TXT 拖拽上传。
- **视频导出**：基于 MediaRecorder API 录制并导出 MP4/WebM 格式。
- **实时预览**：支持不触发录制的纯预览模式。

## 🛠 技术实现

- **纯前端架构**：无需后端，所有计算与渲染均在本地完成。
- **Canvas 2D 绘图**：高性能条形图绘制与粒子物理模拟。
- **响应式布局**：完美支持 9:16 短视频比例，适配多种屏幕尺寸。

## 📁 文件结构
- `index.html`: 页面结构与功能配置面板。
- `css/style.css`: 视觉样式与 UI 布局。
- `js/app.js`: 核心逻辑类 `DynamicRanking`，负责所有交互与动画渲染。
